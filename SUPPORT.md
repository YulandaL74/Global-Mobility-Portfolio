# Support

If you need help or want to ask questions about this project, here are the recommended ways to get support:

- Open a GitHub issue describing your problem, including:
  - What you tried
  - What you expected to happen
  - Any error messages or screenshots
- For feature discussion, use the "feature request" issue template.
- For security-sensitive reports, see SECURITY.md.
